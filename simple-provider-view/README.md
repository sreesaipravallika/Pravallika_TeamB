# 🔍 Service Provider Finder - Simple Version

A beginner-friendly web application to find service providers by location.

## 📁 Files

- `index.html` - Main page structure
- `style.css` - Styling and design
- `script.js` - JavaScript logic
- `EXPLANATION.md` - Detailed explanation

## 🚀 How to Run

1. Open `index.html` in any web browser
2. No installation needed!
3. No server required!

## 🎯 How to Use

1. **Enter a city name** in the search box
   - Try: Mumbai, Delhi, Bangalore, Pune, or Chennai

2. **Click "Search"** button (or press Enter)

3. **View results** - Provider cards will appear showing:
   - Provider Name
   - Brand Name
   - Service Category
   - Service Area
   - Available Timings
   - Price Range

## 🆘 Troubleshooting

**Issue:** Page is blank  
**Solution:** Make sure all 3 files are in the same folder

**Issue:** No styling  
**Solution:** Check that `style.css` filename is correct

**Issue:** Search not working  
**Solution:** Check browser console (F12) for errors

**Issue:** "No services available" always shows  
**Solution:** Make sure you're entering: Mumbai, Delhi, Bangalore, Pune, or Chennai
