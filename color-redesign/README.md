## 🔄 Customization

### Change Primary Color
```css
:root {
    --primary-main: #YOUR_COLOR;
    --primary-hover: #DARKER_SHADE;
}
```

### Adjust Spacing
```css
:root {
    --spacing-md: 1rem;  /* Adjust as needed */
}
```

### Modify Transitions
```css
:root {
    --transition-base: 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
```

## 🆘 Support

### Common Issues

**Colors not showing?**
- Check if CSS file is linked
- Clear browser cache
- Verify class names

**Hover effects not working?**
- Check for CSS conflicts
- Ensure transitions are defined
- Test in different browsers

**Text not readable?**
- Verify contrast ratios
- Use darker text colors
- Check background colors
